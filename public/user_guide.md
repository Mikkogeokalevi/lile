# Käyttöohje (Lile)

## 1. Mikä tämä on?
Lile on lihavuusleikattujen tukiryhmän työkalu, jossa jaetaan:
- Ruokapaikkojen yhteystietoja ja alennuksia
- Reseptejä

Sovellus toimii puhelimella ja voidaan asentaa "sovellukseksi" (PWA).

## 2. Kirjautuminen

### 2.1 Kirjautuminen Google-tilillä
1. Avaa sovellus.
2. Valitse **Kirjaudu**.
3. Paina **Jatka Googlella**.
4. Valitse Google-tili ja hyväksy kirjautuminen.

### 2.2 Kirjautuminen sähköpostilla ja salasanalla
1. Avaa sovellus.
2. Valitse **Kirjaudu**.
3. Syötä **sähköposti** ja **salasana**.
4. Paina **Kirjaudu**.

### 2.3 Uuden tunnuksen luonti (sähköposti + salasana)
1. Avaa sovellus.
2. Valitse **Kirjaudu**.
3. Paina **Luo uusi tunnus**.
4. Syötä sähköposti ja salasana.
5. Paina **Luo tunnus**.

## 3. Toiminnot (yleisnäkymä)

### 3.1 Ruokapaikat
Ruokapaikat-osiossa käyttäjät voivat:
- Lisätä uuden ruokapaikan
- Muokata omia lisäyksiään
- Tykkäillä ruokapaikoista
- Ehdottaa ruokapaikan poistoa (tarkistetaan ylläpidossa)

### 3.2 Reseptit
Reseptit-osiossa käyttäjät voivat:
- Lisätä uuden reseptin
- Muokata omia reseptejään
- Tykkäillä resepteistä
- Ehdottaa reseptin poistoa

## 4. Admin-näkymä
Admin-näkymä näkyy vain ylläpitäjälle. Siellä hallitaan mm. käyttäjiä ja poistoehdotuksia.

## 5. Asentaminen puhelimeen (PWA)

### 5.1 Android (Chrome)
1. Avaa sovellus Chromessa.
2. Paina oikean yläkulman valikkoa (kolme pistettä).
3. Valitse **Lisää aloitusnäyttöön** tai **Asenna sovellus**.
4. Vahvista asennus.

### 5.2 iPhone/iPad (Safari)
1. Avaa sovellus Safarissa.
2. Paina **Jaa**-painiketta (neliö ja nuoli ylöspäin).
3. Valitse **Lisää Koti-valikkoon**.
4. Vahvista.

## 6. Vianetsintä

### 6.1 Kirjautumisikkuna ei aukea (Google)
- Tarkista, ettei selain estä ponnahdusikkunoita.

### 6.2 Sovellus ei lataudu
- Päivitä sivu.
- Tarkista internet-yhteys.
